Do-File Templates

cleaning.do — Data Cleaning 


********************************************************************************
* DATA CLEANING TEMPLATE
* AUTHOR: Dr Abdulla
* DESCRIPTION: Handle missings, outliers, duplicates, and validate data.
********************************************************************************
clear all
set more off
global project_root "C:/Users/YourName/Dropbox/Projects/YourProject"
global data_raw     "$project_root/01_data/raw"
global data_clean   "$project_root/01_data/clean"
global log          "$project_root/04_log"

capture log close
log using "$log/cleaning_`c(current_date)'.log", replace

use "$data_raw/raw_data.dta", clear
missings report
replace income = . if income == -999
winsor2 income age, cuts(1 99) replace
duplicates drop id, force
recode gender (1=0 "Male") (2=1 "Female"), gen(gender_bin)
save "$data_clean/cleaned_data.dta", replace
log close

transformation.do — Reshape & Merge 


********************************************************************************
* DATA TRANSFORMATION TEMPLATE
********************************************************************************
use "$data_clean/cleaned_data.dta", clear
gen log_income = log(income + 1)
egen z_age = std(age)
reshape long sales_, i(id) j(year)
merge 1:1 id year using "$data_clean/covariates.dta", nogen
bysort region: egen mean_income = mean(income)
save "$data_transform/analysis_ready.dta", replace

4-Week Master Do-File

A complete, self-contained workflow combining all webinar and course examples.


* ==============================================================================
* STATA LEARNING: COMPLETE DO-FILE (BEGINNER → ADVANCED)
* ==============================================================================
clear all
set more off

* EXAMPLE 1: DATA IMPORT & CLEANING
import excel "https://stats.idre.ucla.edu/stat/data/hsb2.xls", firstrow clear
rename id ID
label variable read "Reading Score"
gen high_math = (math > 60)
save "hsb2_clean.dta", replace

* EXAMPLE 2: SURVEY DATA
webuse nhanes2f, clear
svyset psuid [pweight=finalwgt], strata(stratid)
svy: mean bpsystol, over(sex)
svy: regress bpsystol age i.sex

* EXAMPLE 3: PANEL DATA
webuse grunfeld, clear
xtset company year
xtreg invest mvalue kstock, fe
estimates store fe
xtreg invest mvalue kstock, re
estimates store re
hausman fe re

* EXAMPLE 4: AUTOMATION & LOOP
eststo clear
foreach var of varlist read write math science {
    eststo: quietly sum `var'
}
esttab using "summary_table.rtf", cells(mean sd) replace
