template_regression.do — Tables


********************************************************************************
* REGRESSION ANALYSIS TEMPLATE
* AUTHOR: Dr Abdulla
********************************************************************************

clear all
set more off

*--- Paths -------------------------------------------------------------------
global data_clean "$project_root/01_data/clean"
global output     "$project_root/03_output"

*--- Load packages -----------------------------------------------------------
foreach pkg in reghdfe estout outreg2 ftools {
    capture which `pkg'
    if _rc ssc install `pkg', replace
}

*--- Load data ---------------------------------------------------------------
use "$data_clean/analysis_data.dta", clear

*--- Run regressions ---------------------------------------------------------
eststo clear
eststo: reg y x1 x2 i.year i.state, vce(cluster district)
eststo: reghdfe y x1 x2, absorb(year state district) vce(cluster district)

*--- Export tables -----------------------------------------------------------
esttab using "$output/regression_table.rtf", ///
    replace b(3) se(3) star(* 0.10 ** 0.05 *** 0.01) ///
    scalars(r2 N) label title("Regression Results") ///
    mtitles("OLS" "FE") ///
    note("SE clustered at district level.")

template_panel.do — Panel Data


********************************************************************************
* PANEL DATA ANALYSIS TEMPLATE
* AUTHOR: Dr Abdulla
********************************************************************************

clear all
set more off

*--- Paths -------------------------------------------------------------------
global data_clean "$project_root/01_data/clean"
global output     "$project_root/03_output"

*--- Load packages -----------------------------------------------------------
foreach pkg in reghdfe xtable {
    capture which `pkg'
    if _rc ssc install `pkg', replace
}

*--- Load data ---------------------------------------------------------------
use "$data_clean/panel_data.dta", clear

*--- Set panel structure -----------------------------------------------------
xtset id year

*--- Fixed Effects -----------------------------------------------------------
eststo: xtreg y x1 x2 i.year, fe vce(cluster id)

*--- Random Effects ----------------------------------------------------------
eststo: xtreg y x1 x2 i.year, re vce(cluster id)

*--- Hausman test ------------------------------------------------------------
hausman fe re

*--- Export ------------------------------------------------------------------
esttab using "$output/panel_table.rtf", replace ///
    b(3) se(3) star(* 0.10 ** 0.05 *** 0.01) ///
    mtitles("FE" "RE") title("Panel Data Results")

twoway (scatter y x, mcolor(navy%60)) ///
       (lfit y x, lcolor(maroon) lwidth(thick)), ///
    title("Y vs X") ///
    xtitle("X") ytitle("Y") ///
    note("Source: Your Data") ///
    saving(scatter.gph, replace)
graph export "$output/scatter.pdf", replace

*--- Coefplot ----------------------------------------------------------------
reg y x1 x2
coefplot, vertical xline(0) ///
    title("Regression Coefficients") ///
    saving(coefplot.gph, replace)

Advanced Jupyter/Stata Notebooks

Interactive Regression + coefplot

Run Stata commands interactively in Jupyter with %%stata magic:

# Install: pip install stata_kernel
# Run: jupyter lab

%%stata
sysuse auto, clear
reghdfe mpg price weight, absorb(rep78) vce(cluster rep78)
estimates store fe
coefplot fe, keep(price) vertical xline(0) ///
    title("Price Effect on MPG") ///
    note("Fixed Effects Model")

Dynamic Panel (Arellano-Bond)

GMM estimation with lags:

%%stata
webuse abdata, clear
xtset id year
ssc install xtabond2, replace
xtabond2 n w k ys L.n, gmm(L.n, lag(2 3)) iv(year) robust
estimates store gmm
esttab gmm using results.rtf, replace
