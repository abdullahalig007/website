34. National Statistics Office (NSO), Ministry of Statistics and Programme Implementation (MoSPI) — Microdata Access Portal

The National Statistics Office (NSO) under MoSPI provides open access to microdata from extensive surveys and censuses through its Microdata Access Portal, powered by the National Data Archive (NADA) software of the International Household Survey Network (IHSN). Researchers, policymakers, and the public can browse, search, and download metadata and unit-level data for over 178 surveys and censuses.

7. GIS Tools

Sl. No Tool Description Link
1 QGIS Open-source GIS software for mapping, spatial analysis, and visualization of agricultural data, climate patterns, and land use. Link
2 ArcGIS Comprehensive GIS platform by Esri for advanced spatial analysis, map creation, and data integration, widely used in agricultural research. Link
3 GRASS GIS Open-source GIS tool for geospatial data management, image processing, and spatial modeling, suitable for environmental and agricultural studies. Link
4 Google Earth Engine Cloud-based platform for analyzing geospatial datasets, including satellite imagery, for climate and agricultural monitoring. Link
5 R with sf/sp Packages R programming language with GIS packages (sf, sp) for spatial data analysis and visualization, ideal for econometric and agricultural studies. Link
6 GeoDa Open-source software for spatial data analysis, exploratory data analysis, and spatial regression, useful for regional agricultural studies. Link
7 GeoFlow A Stata module for generating spatial flow arcs, useful for visualizing and analyzing directional spatial relationships. Link
8 spmap (Stata) A community-contributed Stata command by Maurizio Pisati for drawing choropleth and thematic maps by merging spatial shapefile data (converted via shp2dta or mif2dta) with attribute data. Link
9 geoplot (Stata) A newer Stata command by Ben Jann for drawing maps using frames (in Stata 16+), allowing multiple layers (regions, boundaries, rivers, labels, points) to be combined flexibly. Link
10 tmap (Stata) A suite of Stata programs by Maurizio Pisati for drawing thematic maps — including choropleth, proportional symbol, dot, deviation, and label maps. Suitable for descriptive spatial visualization and regional analysis. Link
11 bimap (Stata) A Stata command for generating bivariate maps that visualize two variables simultaneously. Uses geoplot for Stata 16+ and spmap for older versions. Link
